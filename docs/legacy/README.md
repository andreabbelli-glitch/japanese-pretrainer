# Archivio Legacy LLM

> [!IMPORTANT]
> Questa cartella e solo archivio storico. Umani, agent e LLM non devono usare
> `docs/legacy/` per istruzioni operative correnti, prompt correnti o template
> correnti.
>
> Per il workflow attuale con LLM esterni bisogna partire sempre da
> `docs/llm-kit/README.md`.

## Scopo

Qui vivono documenti storici mantenuti solo per:

- conservare contesto e decisioni precedenti;
- evitare perdita di riferimenti usati in note o task storici;
- confrontare versioni vecchie del workflow.

## Regola Operativa

Se un file in `docs/legacy/` contraddice il kit LLM o altri documenti correnti,
vale sempre il kit:

- `docs/llm-kit/README.md`
- `docs/llm-kit/general/*`
- `docs/llm-kit/media/<media-slug>/*`

## Contenuto

- `llm-content-handoff.md`
- `content-workflow-playbook.md`
- `content-briefs/`
- `prompts/`
- `templates/`
