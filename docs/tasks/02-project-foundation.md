# Task 02 - Project Foundation

## Tipo

Platform setup

## Obiettivo

Creare lo scheletro applicativo e gli strumenti base di sviluppo.

## Scope

- inizializzare app Next.js + TypeScript;
- configurare struttura cartelle iniziale;
- configurare linting, formatting e test runner;
- impostare variabili ambiente locali;
- predisporre script per dev, build e check;
- aggiungere base CSS e token placeholder.

## Deliverable

- repository avviabile in locale con `npm` o `pnpm`;
- cartelle iniziali per app, db, lib, content, tests;
- config di lint e test funzionanti;
- pagina placeholder con shell minima;
- README aggiornato con istruzioni di bootstrap.

## Dipendenze

- nessuna

## Criteri di accettazione

- `dev`, `build`, `lint` e `test` girano in locale;
- la struttura del progetto e coerente con i task successivi;
- il progetto puo essere aperto e avviato senza setup esterno oltre a Node.

## Note

Non implementare logica di dominio in questo task. L'obiettivo e preparare una
base pulita per persistence, parser e UI.
