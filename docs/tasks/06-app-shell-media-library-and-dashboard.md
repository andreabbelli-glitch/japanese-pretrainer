# Task 06 - App Shell, Media Library And Dashboard

## Tipo

Frontend / navigation

## Obiettivo

Costruire lo shell dell'app e le pagine di ingresso al sistema di studio.

## Input

- `01`
- `02`
- `03`

## Scope

- layout globale responsive;
- navigazione primaria;
- homepage/dashboard;
- media library;
- media detail page con overview;
- pattern base per card, section, tabs e sticky header.

## Deliverable

- shell coerente desktop/mobile;
- dashboard con stato vuoto e stato popolato;
- lista media con metriche essenziali;
- pagina dettaglio media con entry point verso textbook, glossary, review e progress.

## Dipendenze

- `01`
- `02`
- `03`

## Criteri di accettazione

- l'utente capisce subito da dove iniziare;
- la navigazione tra aree principali richiede pochi tocchi/click;
- il layout e gradevole e non sembra un pannello amministrativo;
- desktop e mobile usano pattern diversi quando serve, non solo resize.

## Note UX

La dashboard deve trasmettere calma e focus. Mostrare pochi indicatori utili, non
un muro di metriche.
